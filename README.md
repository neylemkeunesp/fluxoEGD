# FluxoEGD

Projeto LaTeX contendo diagrama de fluxograma.

## Conteúdo

- `main.tex` - Documento LaTeX principal
- `fluxo_original.png` - Imagem do fluxograma original
- `main.pdf` - Documento compilado

## Como compilar

Para gerar o PDF a partir do arquivo LaTeX:

```bash
pdflatex main.tex
```
